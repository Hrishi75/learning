# AWS Interview Questions And Answers

This folder is for interview preparation. The answers are written from a
DevOps/cloud engineering point of view and are grounded in official AWS and
HashiCorp documentation.

## How To Study

1. Read one file per day.
2. Say each answer out loud in your own words.
3. For scenario questions, explain tradeoffs, not just service names.
4. When asked "why", mention reliability, security, cost, operations, and scale.
5. After reading an answer, run one related CLI command if possible.
